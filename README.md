### 上手指南

欢迎各位延大学子参与开发与BUG修正

###### 开发前的配置要求

1. Android Studio
2. Jvm 1.8
3. compose_version   '1.2.0'

###### **安装步骤**

Clone the repo

```sh
git clone https://github.com/eventhorizonsky/YBUtools.git
```
